About the Class
This is an introductory web design course. Students will develop basic knowledge of the internet, html, css, WordPress, Squarespace and other web design/publishing tools. The instructor will provide some demonstrations of concepts and techniques, but the majority of course work will involve students learning by doing.

In this class you will:

  1. learn the difference between CSS and HTML
  2. create your own website using HTML
  3. learn the coding lanugage

Class will meet on these days and times:

  • Tuesdays- 1:30PM-2:45PM
  • Thursdays- 1:30PM-2:45PM

Attendance is mandatory
