My Journey to Web Design!

About

Welcome to my website! Over the past few months, I have had the privelage of learning different coding and web design programs through my MediaPro 303 class at Boise State University. Usually, my focus is on video and photography work on a camera or on a computer. It was not until this course that I was fully able to understand the basics of buidling a website that could potentially highlight these. Not only have I learned the basic foundations of HTML and CSS coding processes, but I also experimented with various outside sites such as Squarespace and Wordpress. Although these sites are pre-formatted, it was still a great skill to utilize for the future, or my future employer.

Before continuing to learn about my journey and how I progressed in web design, it is important to know the basic foundations about what the internet is and how it works. Basic Tools to Get You Started!

There were a few different tools I was able to use in the duration of the course to help guide me through my work. The first was FreeCodeCamp. This website offers super use4r-friendly step-by-step challenges to teach you different coding basics. I completed the "Legacy Responsive Web Design Certification", but they offer a variety of different certificates depending on what area of focus you want to learn about!

Link to FreeCodeCamp Website!

The second tool we used was W3 Schools. Once I had gotten a solid idea of how to code, I was able to reference W3 Schools to remember exactly how to format the coding I wanted. Because there is so much information involving coding, W3 is a great resource to refer back to on skills you have already learned. Along with this, it has exact coding examples and explains how to format them if you are needing extra help or want to try something new!
